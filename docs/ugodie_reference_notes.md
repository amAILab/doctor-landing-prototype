# UX-референс: клиника «Угодие»

Источник: https://ugodie.ru/  
Дата просмотра: 2026-05-09

## Что Николаю нравится / что можно адаптировать

Не копировать тексты и дизайн. Использовать как UX-логику.

### Паттерны

1. Сразу понятно, что это наркологическая помощь.
2. На первом экране есть быстрый телефон и вызов врача.
3. Услуги оформлены карточками с ценой и кнопкой.
4. Есть блок принципов работы: качество, доверие, опыт, индивидуальный подход, анонимность/конфиденциальность, удобство.
5. Есть акцент на врачей/специалистов.
6. Есть блок лицензий, сертификатов, документов.
7. Есть простой сценарий получения помощи в несколько шагов.
8. Есть круглосуточность/быстрый контакт.
9. Есть предупреждение о лицензированности и медицинском контуре.

## Что НЕ переносить без проверки

- Конкретные цены «Угодия».
- Гарантии анонимности в абсолютной форме.
- Утверждения о стаже, лицензии, клинике, врачах, палатах, препаратах.
- Названия методов/препаратов без юридической и медицинской проверки.
- Любые чужие формулировки, отзывы, фото, сертификаты.

## Что добавить в MVP «Доктор»

- Верхняя строка: телефон-заглушка, 24/7 после подтверждения, цена от 15 000 ₽ + такси.
- Карточки услуг с ценами: консультация, выезд, помощь семье, сопровождение.
- Блок «Принципы работы»: конфиденциально, спокойно, без давления, стоимость заранее, экстренные случаи в 103/112.
- Блок «Документы»: место под лицензию/реквизиты/политику персональных данных.
- Блок «Николай Курников»: персональный контакт, без неподтверждённых регалий.
- CTA повторять после важных блоков.
