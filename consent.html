Согласие на обработку персональных данных

Отправляя форму на сайте PRO Жизнь или звоня по указанному номеру, пользователь подтверждает согласие на обработку данных для обратной связи по обращению.

1. Данные, на которые даётся согласие

Пользователь может передать имя, телефон, район выезда, срочность обращения и краткое описание ситуации. Пользователь самостоятельно определяет объём информации, которую сообщает.

2. Цели обработки

  • связаться с пользователем;
  • уточнить ситуацию и срочность обращения;
  • согласовать возможный формат помощи;
  • сообщить, если по описанию ситуации необходимо обращаться в экстренную службу 103/112.

3. Действия с данными

Согласие распространяется на сбор, запись, систематизацию, хранение, уточнение, использование, передачу уполномоченным лицам для обработки обращения, обезличивание, блокирование и удаление данных.

4. Срок действия и отзыв согласия

Согласие действует до достижения целей обработки или до его отзыва пользователем. Отозвать согласие можно через тот канал связи, по которому пользователь обращался, либо по актуальному контакту, указанному владельцем сайта.

5. Важные ограничения

Форма сайта не предназначена для передачи избыточных медицинских документов, паспортных данных, точного адреса и иной чувствительной информации, не нужной для первичной связи.

Сайт не является службой экстренной помощи. При угрозе жизни и здоровью необходимо звонить 103 или 112.
