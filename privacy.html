Политика обработки персональных данных

Эта страница описывает минимальный порядок обработки данных, которые пользователь оставляет для обратной связи.

1. Какие данные собираются

Форма может собирать имя, телефон, район выезда, срочность обращения и краткое описание ситуации.

2. Зачем нужны данные

Данные используются только для обратной связи, уточнения ситуации и согласования возможного формата помощи.

3. Медицинская оговорка

Сайт не является службой экстренной медицинской помощи и не заменяет очную консультацию врача. При угрозе жизни и здоровью необходимо обращаться по телефонам 103 или 112.

4. Передача данных

Данные не публикуются на сайте и используются только для связи по обращению. Доступ к ним должен быть ограничен лицами, которые обрабатывают заявку.

5. Права пользователя

  • уточнить, какие данные были переданы;
  • попросить удалить или исправить контактные данные;
  • отозвать согласие на дальнейшую обработку обращения;
  • получить информацию об операторе данных через согласованный канал связи.
