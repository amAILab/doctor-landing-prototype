Политика обработки персональных данных

Документ описывает, какие данные может получить сайт PRO Жизнь, зачем они нужны и как пользователь может отозвать согласие.

1. Общие положения

Сайт используется для первичной обратной связи по вопросам конфиденциальной наркологической помощи. Обработка данных выполняется в объёме, необходимом для связи с пользователем, уточнения обращения и согласования возможного формата помощи.

До размещения сайта на постоянном домене сведения об операторе персональных данных, юридическом лице/ИП, адресе, лицензии и актуальных контактах должны быть дополнены владельцем сайта.

2. Какие данные могут обрабатываться

  • имя или удобное обращение;
  • номер телефона;
  • район или ориентир выезда;
  • срочность обращения;
  • краткое описание ситуации, которое пользователь вводит самостоятельно;
  • технические данные посещения сайта, если они собираются хостингом или системами аналитики.

Пользователю не следует отправлять паспортные данные, точный адрес, фотографии документов, подробные медицинские документы и иную избыточную информацию через форму сайта.

3. Цели обработки

  • обратная связь по обращению;
  • уточнение срочности и общего характера ситуации;
  • согласование возможной консультации, выезда специалиста или рекомендации обратиться в экстренную службу;
  • исполнение требований законодательства и защита прав сторон при спорных ситуациях.

4. Правовое основание

Данные обрабатываются на основании согласия пользователя, выраженного при отправке формы или при самостоятельном звонке по указанному номеру. Пользователь может не оставлять данные и вместо этого обратиться в экстренные службы при угрозе жизни или здоровью.

5. Срок хранения

Данные хранятся только столько, сколько необходимо для обработки обращения и последующей связи, если иной срок не требуется по закону или для защиты прав сторон. По запросу пользователя данные могут быть удалены, если нет законных оснований для дальнейшего хранения.

6. Передача третьим лицам

Данные не публикуются на сайте и не используются для открытого размещения. Доступ к данным должен быть ограничен лицами, которые обрабатывают обращение. Передача третьим лицам возможна только при необходимости организации помощи, по требованию закона или с согласия пользователя.

7. Медицинская оговорка

Сайт не является службой экстренной медицинской помощи и не заменяет очную консультацию врача. При угрозе жизни, потере сознания, судорогах, нарушении дыхания, боли в груди, травме или тяжёлом состоянии необходимо звонить 103 или 112.

Лечение и медицинское вмешательство возможны только с согласия пациента, за исключением случаев, прямо предусмотренных законом.

8. Права пользователя

  • уточнить, какие данные были переданы;
  • попросить исправить контактные данные;
  • отозвать согласие на дальнейшую обработку;
  • попросить удалить данные, если нет законных оснований для хранения;
  • получить информацию об операторе данных через согласованный канал связи.
